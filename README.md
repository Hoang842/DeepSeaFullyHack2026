# Abyss Hero Game

## Project Overview

**Abyss Hero** is a Unity-based game where players explore an underwater environment to save marine animals and clean up ocean pollution. Players use tools like scissors to free trapped animals and a mesh bag to collect trash, all while managing a limited oxygen supply.

The game aims to promote environmental awareness through interactive gameplay and engaging mechanics.

---

## Features

* Cut animals free from debris using scissors
* Collect trash with a mesh bag
* Oxygen system that depletes over time
* Interactive tutorial system with looping demonstrations
* Underwater-themed visuals and UI

---

## Built With

* **Engine:** Unity
* **Language:** C#
* **Tools:** Unity Timeline, UI System (Canvas), Render Textures, Procreate

## Gameplay Mechanics

### Saving Animals

Players locate trapped marine animals and use scissors to cut them free from debris.

### Trash Collection

Players use a mesh bag to collect floating trash throughout the ocean.

### Oxygen System

Oxygen gradually decreases over time, requiring players to manage their exploration carefully.

---

## Tutorial System

The game includes an in-game tutorial with looping visual demonstrations of:

* Cutting animals free
* Collecting trash
* Managing oxygen levels
* Interacting with enemies

These are implemented using Unity Timeline and displayed in UI panels.
